Detailed Description

Declares the EasyEaseAnimator template class for advanced animation easing.

This file defines the EasyEaseAnimator class, which implements the IEasyEaseAnimator interface and provides functionalities for smooth and customizable animations using various interpolation methods, damped springs, and ramp filters.
